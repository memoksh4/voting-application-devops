## Overview
This project is a simple web-based Voting Application developed using Python Flask and implemented with core DevOps practices. The application allows users to vote for candidates and view live results through a web interface.

## Objective
The main objective of this project is to demonstrate hands-on DevOps skills by building, containerizing, and automating a real working application using industry-standard tools.

## Technologies Used
- Python (Flask)
- HTML
- Docker
- Git & GitHub
- GitHub Actions (CI)

## DevOps Implementation
- The application is containerized using Docker for consistent and portable deployment.
- A CI pipeline is implemented using GitHub Actions, which automatically builds the Docker image on every code push.
- Version control is managed using Git and hosted on GitHub.

## Outcome
The project successfully demonstrates end-to-end DevOps workflow, including application development, containerization, version control, and continuous integration, making it suitable for entry-level DevOps and cloud roles.
